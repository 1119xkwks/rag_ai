# rag_core

## 프로젝트 소개
`rag_core`는 Python 기반의 Gemini 챗봇 및 RAG(Retrieval-Augmented Generation) AI 백엔드 프로젝트입니다.
Google DeepMind의 Gemini 모델을 활용하여 질의응답 시스템을 구축하고, RAG 기술을 통해 외부 지식 기반의 정확도 높은 답변을 제공하는 것을 목표로 합니다.

## 주요 기능
* **Gemini Chatbot:** Google Gemini API를 연동한 대화형 AI 챗봇
* **RAG AI Backend:** 문서 검색 및 생성을 결합한 RAG 파이프라인 구현
* **Python 기반:** 확장성과 유지보수가 용이한 Python 환경

## 시작하기

### 필수 조건
* Python 3.8 이상
* Gemini API Key

### 설치 방법

1. 저장소를 클론합니다.
   ```bash
   git clone <repository_url>
   cd rag_core
   ```

2. 가상 환경을 생성하고 활성화합니다.
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # macOS/Linux
   source venv/bin/activate
   ```

3. 의존성 패키지를 설치합니다.
   ```bash
   pip install -r requirements.txt
   ```
